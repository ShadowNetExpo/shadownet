Publicar en Vercel

Tu app en internet
en 10 minutos

Ya tienes cuenta en Vercel. Ahora sigue exactamente estos pasos.

1

Descarga el código de ShadowNet

El archivo ShadowNet.jsx que construimos es tu app. Crea una carpeta en tu computadora llamada shadownet y colócalo ahí.

2

Abre la terminal en esa carpeta y ejecuta:

npx create-next-app@latest . --typescript

Luego: npm run dev → visita localhost:3000 para ver que funciona.

3

Sube el código a GitHub

git init → git add . → git commit -m "ShadowNet v1" → Crea repositorio en github.com y sigue las instrucciones que te da GitHub.

4

Importa desde Vercel

vercel.com → "Add New Project" → selecciona tu repositorio de GitHub → "Deploy". En 2 minutos tienes una URL en vivo del tipo shadownet.vercel.app.

5

Conecta tu dominio propio

Vercel → Settings → Domains → escribe shdwnet.app → Add. Vercel te dice exactamente qué configurar en Namecheap (2 registros DNS).

✅

¡Tu app estará en vivo!

Cada vez que hagas cambios en el código y subas a GitHub (git push), Vercel actualiza la app en vivo automáticamente. Sin reinstalar nada.
